# TASKS.md
# 漫享家 / MangaHaven 分批任务清单

## 使用规则
- 每次只做一个阶段
- 每阶段都必须可运行
- 每阶段都要附带验收步骤
- 不允许一次性做完所有功能
- 优先保证架构和稳定性，不追求表面完整

---

# Phase 0：工程骨架

## 目标
建立一个可扩展、可维护、可逐步演进的 Android 原生项目。

## 任务清单

### P0-1 工程初始化
- [ ] 创建 Android 项目
- [ ] 配置 Kotlin、Compose、Material 3
- [ ] 配置多模块结构
- [ ] 配置基础 Gradle Convention 或统一依赖管理
- [ ] 配置 debug / release 构建类型

### P0-2 模块搭建
- [ ] 创建 `app`
- [ ] 创建 `core-model`
- [ ] 创建 `core-reader`
- [ ] 创建 `data-local`
- [ ] 创建 `data-files`
- [ ] 创建 `feature-library`
- [ ] 创建 `feature-reader`
- [ ] 创建 `feature-settings`

### P0-3 基础设施
- [ ] 接入 Room
- [ ] 接入 DataStore
- [ ] 接入 WorkManager
- [ ] 接入 DI（Hilt 或 Koin）
- [ ] 接入日志系统
- [ ] 统一 Result / Error 模型

### P0-4 UI 骨架
- [ ] 建立 AppNavigation
- [ ] 建立书架页空壳
- [ ] 建立阅读器页空壳
- [ ] 建立设置页空壳
- [ ] 建立主题系统（浅色 / 深色）

### P0-5 数据骨架
- [ ] 定义 Source 实体
- [ ] 定义 LibraryItem 实体
- [ ] 定义 ReadingProgress 实体
- [ ] 定义 ReaderSettings 数据结构
- [ ] 建立 Room DAO 雏形

## 验收标准
- [ ] 项目可编译运行
- [ ] 三个页面可导航切换
- [ ] DataStore 可保存测试设置
- [ ] Room 可写入测试数据
- [ ] WorkManager 可跑一个示例任务
